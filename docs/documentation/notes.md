# Notes

The Notes feature provides an area to to write down and view any useful info or reminders. Notes are supported on products and shops.

## View Product Note 

To view a product note, long-press the product in the [Product List](/docs/documentation/product-list), then tap the *Note* icon ![Note icon](/assets/images/screenshots/light-mode/alr-995-view-note.png) 
in the header bar.
{: .v-align-middle }

## View Shop Note 

To view a shop note, long-press the shop on the [All Shops Page](/docs/documentation/manage-shops#all-shops-page), then tap the *Note* icon ![Note icon](/assets/images/screenshots/light-mode/alr-995-view-note.png) in the header bar.
{: .v-align-middle }

## Note Dialog

![Note Dialog](/assets/images/screenshots/light-mode/alr-330-notes-dialog-partial.png)
{: .mx-auto .float-md-right }

Tapping the *Note* icon on a product or shop will display the note dialog. On the note dialog, the note can be viewed, edited or deleted. 

To add or edit a note, type the note into the text box and tap *Ok*.

To remove a note, clear the text box and Tap *Ok*.

Tap *Cancel* to discard any changes and keep the existing note.
